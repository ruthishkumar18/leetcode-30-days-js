# Day 20 — 2727. Is Object Empty

## Problem
Given an object or an array, return true if it is empty.

- An empty object has no key-value pairs.
- An empty array has no elements.

The input is the result of JSON.parse.

## Examples
- {"x": 5} → false
- {} → true
- [null, false, 0] → false

## Constraints
- obj is a valid JSON object or array
