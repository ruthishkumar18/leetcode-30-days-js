# Notes — Day 01

### Approach
The task is simple: return a function that always outputs `"Hello World"`.

### Key Concepts
- Closures in JavaScript
- Functions returning other functions
- Ignoring parameters using rest operator `(...args)`

### Complexity
- Time: **O(1)**
- Space: **O(1)**

### Takeaway
A good warm‑up problem to recall how closures and function execution work in JavaScript.
