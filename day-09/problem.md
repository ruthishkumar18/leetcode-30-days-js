# Day 09 — 2703. Return Length of Arguments Passed

## Problem
Write a function `argumentsLength` that returns the number of arguments passed to it.

## Examples

### Example 1
Input: [5]  
Output: 1  
Explanation: argumentsLength(5) → 1

### Example 2
Input: [{}, null, "3"]  
Output: 3  
Explanation: argumentsLength({}, null, "3") → 3

## Constraints
- args is a valid JSON array  
- 0 <= args.length <= 100
